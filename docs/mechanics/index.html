Game Mechanics Wiki

A reference for every system in Project Chimera — from the 100ms combat tick to the campaign overworld. Pick a topic to dive in.

Project Chimera is a three-layer tactical simulation. Each layer feeds into the next:

Campaign (turn-based overworld)
  ↓ launches
Mission (multi-room dungeon)
  ↓ enters
Combat (100ms fixed-tick deterministic sim)
  ↓ resolves via
step(state, intents, dt_ms) → (state, events)

Every mechanic documented below lives in one of these layers. Combat is the deepest and most detailed.
