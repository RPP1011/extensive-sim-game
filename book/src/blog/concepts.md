# ML Concepts

A six-part learning path covering the mathematical and algorithmic foundations
used throughout the project's ML pipeline. Each guide builds on the previous one.

> These concept guides are available as standalone rich HTML pages with custom
> styling. Click through for the full reading experience.

## Learning Path

### 1. [Foundations](../../OLD/concepts_foundations.html)
Neural networks, gradient descent, loss functions, batch/epoch/LR, overfitting,
normalization.

### 2. [Math](../../OLD/concepts_math.html)
Probability, expectation, log probability, entropy, KL divergence, dot product,
matrix multiplication, cosine similarity, EMA.

### 3. [Architecture](../../OLD/concepts_architecture.html)
Tokenization, residual connections, autoencoders, latent spaces.

### 4. [Attention](../../OLD/concepts_attention.html)
Self-attention, cross-attention, logits & softmax, masked language modeling,
scaled dot-product attention.

### 5. [RL Foundations](../../OLD/concepts_rl_foundations.html)
Actor-critic, value & Q-functions, on-policy vs off-policy, reward shaping,
replay buffer.

### 6. [RL Algorithms](../../OLD/concepts_rl_algorithms.html)
REINFORCE, PPO, IMPALA & V-Trace, SAC.
